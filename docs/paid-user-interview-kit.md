# 유료 사용자 10명 인터뷰 운영 키트

정책알리미 유료 전환자가 **왜 결제했고, 어디서 막히며, 계속 쓸 이유가 무엇인지** 10분 안에 확인하기 위한 실행 문서입니다.

## 목적

- 가격표·checkout·결제 직후 활성화 UX가 실제로 이해되는지 확인한다.
- Basic/Pro 각각에서 사용자가 기대하는 핵심 가치를 분리한다.
- 해지 위험 신호와 다음 개선 우선순위를 찾는다.
- 광고/콘텐츠에 바로 쓸 수 있는 실제 표현을 수집한다.

## 모집 기준

| 그룹 | 목표 인원 | 우선순위 | 확인하고 싶은 것 |
|---|---:|---|---|
| Basic 결제자 | 5명 | 1순위 | 사업자 정보·이메일 알림·자격 진단 가치 |
| Pro 결제자 | 5명 | 1순위 | 카카오 알림톡·AI 상담·신청서 초안 기대 |
| 결제 후 미설정 사용자 | 포함 권장 | 2순위 | 결제 후 무엇을 해야 할지 막힌 지점 |
| checkout 실패 후 재시도 사용자 | 가능하면 1~2명 | 3순위 | 실패 안내와 재시도 흐름 |

## 인터뷰 원칙

- 설명하지 말고 먼저 물어본다.
- 칭찬을 유도하지 않는다. “좋죠?” 대신 “어디가 애매했나요?”로 묻는다.
- 기능 선호보다 **실제 행동**을 묻는다.
- 사용자의 단어를 그대로 기록한다.
- 10분을 넘기지 않는다. 더 길어지면 후속 인터뷰로 분리한다.

## 10분 진행표

| 시간 | 구간 | 목표 |
|---:|---|---|
| 0:00~0:30 | 인사·동의 | 녹음/기록 동의, 짧은 안내 |
| 0:30~2:00 | 결제 계기 | 왜 결제했는지 파악 |
| 2:00~4:00 | 가격표·결제 흐름 | 망설임/불안/이해도 확인 |
| 4:00~6:30 | 결제 직후 행동 | 어떤 기능을 켰는지, 어디서 멈췄는지 확인 |
| 6:30~8:30 | 핵심 가치·해지 위험 | 계속 쓸 이유와 해지 이유 찾기 |
| 8:30~10:00 | 한 줄 추천·마무리 | 카피/증언 소재 수집 |

## 시작 멘트

```text
안녕하세요. 정책알리미를 직접 써보신 분들께 10분만 짧게 여쭤보고 있어요.
좋은 말만 들으려는 게 아니라, 어디가 헷갈렸는지 찾는 게 목적입니다.
말씀해주신 내용은 서비스 개선용으로만 정리하고, 이름이나 개인정보는 공개하지 않을게요.
편하게 솔직하게 말씀해주세요.
```

## 핵심 질문지

### 1. 결제 이유

1. 처음에 어떤 문제를 해결하려고 정책알리미를 보셨나요?
2. 무료로 보다가 결제해야겠다고 느낀 순간이 있었나요?
3. Basic/Pro 중 지금 플랜을 고른 이유는 무엇인가요?
4. 결제 전 마지막으로 망설인 이유가 있었다면 무엇인가요?

### 2. 가격표·checkout 이해도

1. 가격표에서 “이 플랜을 쓰면 무엇을 얻는지” 바로 이해됐나요?
2. 헷갈리거나 과장처럼 느껴진 문구가 있었나요?
3. checkout에서 카드 등록·7일 무료체험·해지 조건이 충분히 안심됐나요?
4. 결제 버튼을 누르기 전 가장 불안했던 점은 무엇인가요?

### 3. 결제 직후 활성화

1. 결제 후 바로 무엇을 해야 할지 알겠던가요?
2. 실제로 가장 먼저 누른 기능은 무엇인가요?
3. 사업자 정보, 알림 조건, 카카오 동의 중 아직 안 한 것이 있다면 이유가 무엇인가요?
4. 결제 후 “아, 이걸 먼저 해야 하는구나”라고 느끼게 만든 안내가 있었나요?

### 4. 핵심 가치

1. 지금까지 가장 유용했던 기능은 무엇인가요?
2. “이 기능 때문에 돈 낼 만하다”고 느낀 부분이 있나요?
3. 반대로 없어도 될 것 같은 기능은 무엇인가요?
4. 주변 사장님/지인에게 설명한다면 한 문장으로 어떻게 말하시겠어요?

### 5. 해지 위험

1. 한 달 뒤 해지한다면 가장 큰 이유는 무엇일까요?
2. 계속 쓰게 만들려면 무엇이 꼭 개선되어야 할까요?
3. 알림이 너무 적거나 많다고 느낀 적이 있나요?
4. 가격은 현재 가치 대비 어떤 느낌인가요? 싸다/적당하다/비싸다 중 어디에 가까운가요?

## 플랜별 추가 질문

### Basic 사용자

```text
사업자 정보를 입력하면 자격 진단이 더 정확해진다는 점이 충분히 전달됐나요?
이메일 알림만으로도 충분하다고 느끼나요, 아니면 카카오 알림이 필요하다고 느끼나요?
```

### Pro 사용자

```text
카카오 알림톡이 Pro를 고른 이유였나요?
AI 상담이나 신청서 초안 기능은 실제로 써볼 생각이 드나요?
Pro 가격이 납득되려면 어떤 결과를 받아야 하나요?
```

### 결제 후 미설정 사용자

```text
결제 후 사업자 정보/카카오 동의/알림 조건 설정을 안 하신 이유가 무엇인가요?
시간이 없어서였나요, 필요성을 못 느껴서였나요, 화면에서 못 찾아서였나요?
```

## 섭외 메시지 템플릿

### 결제 직후 사용자용

```text
안녕하세요, 정책알리미 운영자입니다.
최근 유료 플랜을 시작해주셔서 정말 감사합니다.
서비스를 더 쉽게 만들기 위해 10분 정도 짧게 통화로 의견을 듣고 있어요.
좋은 말보다 “헷갈렸던 점·아쉬웠던 점”을 듣고 싶습니다.
가능하신 시간 알려주시면 맞춰서 연락드리겠습니다.
```

### 미설정 사용자용

```text
안녕하세요, 정책알리미 운영자입니다.
유료 플랜을 시작하신 뒤 아직 일부 설정이 완료되지 않은 것 같아,
혹시 어떤 부분이 막혔는지 10분만 여쭤보고 싶습니다.
설명이 부족했는지, 필요성을 못 느끼셨는지 확인해서 바로 개선하려고 합니다.
가능하신 시간 편하게 알려주세요.
```

### checkout 실패 사용자용

```text
안녕하세요, 정책알리미 운영자입니다.
카드 등록 과정에서 불편을 겪으신 것 같아 죄송합니다.
실패 안내나 재시도 흐름이 충분했는지 5~10분 정도만 여쭤봐도 될까요?
결제 유도 목적이 아니라, 막힌 지점을 고치기 위한 확인입니다.
```

## 인터뷰 후 분류 기준

| 분류 | 기준 | 다음 액션 |
|---|---|---|
| 가격표 문제 | 결제 전 가치 이해 실패 | 가격표 문구/플랜 비교 개선 |
| checkout 불안 | 카드·무료체험·해지 조건 불안 | checkout 안심 문구 보강 |
| 활성화 실패 | 결제 후 설정 미완료 | `/mypage` 리마인더/온보딩 보강 |
