# 정부 정책 API 키 발급 가이드

keepioo 에 정부·지자체 지원 정책을 자동으로 수집하려면 아래 3곳에서 **API 키(비밀번호 같은 식별자)** 를 발급받아야 합니다. 모두 무료이고, 회원가입 후 10분 이내로 받을 수 있어요.

---

## 준비물

- 본인 이메일 (Gmail·네이버 등)
- 인증 가능한 휴대폰 번호

---

## 1. 공공데이터포털 (data.go.kr) — 제일 중요

keepioo 가 사용하는 **7가지 정부 API 를 하나의 키로 전부 쓸 수 있어요**. 가장 먼저 발급받으세요.

### 1-1. 회원가입
1. https://www.data.go.kr 접속
2. 오른쪽 상단 **로그인 → 회원가입** 클릭
3. 이메일·비밀번호 입력 → 이메일 인증 → 로그인

### 1-2. API 7개 활용신청
로그인한 상태에서 아래 링크를 하나씩 열고, 페이지 오른쪽 위 **"활용신청"** 초록 버튼을 누르세요.

활용신청 창이 뜨면:
- **활용목적**: `keepioo.com 에서 이용자들에게 맞춤 정책 알림을 제공하기 위함`
- **동의** 체크 모두 → **신청** 버튼

신청할 API 7개:

| 번호 | API 이름 | 링크 |
|------|---------|------|
| ① | 행정안전부 대한민국 공공서비스(혜택) 정보 — **보조금24** | https://www.data.go.kr/data/15113968/openapi.do |
| ② | 복지로 중앙 — 국가 복지 | https://www.data.go.kr/data/15062335/openapi.do |
| ③ | 지자체 복지 — 지방자치단체 복지 | https://www.data.go.kr/data/15083323/openapi.do |
| ④ | 소상공인 지원사업 (MSS) | https://www.data.go.kr/data/15064479/openapi.do |
| ⑤ | 창업진흥원 K-Startup 조회서비스 | https://www.data.go.kr/data/15125364/openapi.do |
| ⑥ | 중소벤처기업부 사업공고 | https://www.data.go.kr/data/15113297/openapi.do |
| ⑦ | 한국고용정보원 워크넷 정부지원일자리정보 | https://www.data.go.kr/data/3071369/openapi.do |

모두 **자동승인** 또는 **1영업일 내 승인** 됩니다.

### 1-3. 키 확인
1. 로그인 상태에서 오른쪽 상단 **마이페이지 → 오픈 API → 활용신청 현황** 클릭
2. 승인된 항목 아무거나 클릭
3. 화면에 표시되는 **"일반 인증키(Encoding)"** 값 복사
   - 예: `a1B2c3D4e5F6... ` 형태의 긴 문자열
   - ⚠️ **"Decoding" 말고 "Encoding"** 이라고 적힌 것 복사

> **좋은 소식**: 7개 API 를 신청해도 **인증키는 계정당 1개**입니다. 하나 복사해서 다 쓸 수 있어요.

### 1-4. 환경변수 등록

로컬(개발용)과 Vercel(서비스용) **두 군데 모두** 등록해야 합니다.

**Vercel**:
1. https://vercel.com/dashboard 접속 → `keepioo` 프로젝트 클릭
2. **Settings → Environment Variables** 이동
3. 다음과 같이 추가:
   - Name: `DATA_GO_KR_API_KEY`
   - Value: 위에서 복사한 인증키 (Encoding 버전)
   - Environments: **Production, Preview, Development 모두 체크**
4. **Save** 클릭

**로컬** (이미 있을 수도 있음):
프로젝트 폴더의 `.env.local` 파일에 아래 줄이 있는지 확인. 없으면 추가.
```
DATA_GO_KR_API_KEY=여기에_붙여넣기
```

## 2. 기업마당 (bizinfo.go.kr)

중소벤처기업부가 운영하는 포털. 매일 **1,400개 이상의 최신 지원사업** 을 받을 수 있어요.

### 2-1. 회원가입
1. https://www.bizinfo.go.kr 접속
2. 오른쪽 상단 **회원가입** → 정보 입력

### 2-2. API 키 발급
1. 로그인 후 https://www.bizinfo.go.kr/apiList.do 접속
2. **"지원사업정보 API"** 항목에서 **"사용신청"** 클릭
3. 간단한 신청서 작성:
   - 활용목적: `맞춤 지원사업 알림 제공`
   - 동의 체크 → 신청
4. 보통 **즉시 승인** 됨
5. **마이페이지 → API 이용신청 현황** 에서 발급된 **서비스 키(crtfcKey)** 복사

### 2-3. 환경변수 등록
Vercel + 로컬 `.env.local` 두 곳에 추가:
```
BIZINFO_API_KEY=여기에_붙여넣기
```

## 3. 온통청년 (youthcenter.go.kr)

국무조정실 청년정책조정실이 운영. 청년 맞춤 정책에 필수예요.

### 3-1. 회원가입
1. https://www.youthcenter.go.kr 접속
2. 오른쪽 상단 **회원가입**
3. 일반 회원으로 가입 (본인인증 필요)

### 3-2. API 키 발급
1. 로그인 후 **마이페이지 → OPEN API** 메뉴 이동
   - 직접 링크: https://www.youthcenter.go.kr/myPage/openapi
2. **"OPEN API 신청"** 버튼 클릭
3. 신청서 작성:
   - 사용구분: **개발용** (또는 운영용)
   - OPEN API 명: **청년정책 API**
   - 사용목적: `keepioo.com 청년 맞춤 정책 알림 제공`
4. 신청 후 보통 **1~3영업일 이내 승인** (관리자 검토)

### 3-3. 키 확인 & 등록
승인되면 **마이페이지 → OPEN API** 에 **인증키** 가 표시됩니다.

Vercel + 로컬 `.env.local` 두 곳에 추가:
```
YOUTH_CENTER_API_KEY=여기에_붙여넣기
```

## 4. 전체 환경변수 요약

다 등록하고 나면 `.env.local` 및 Vercel Environment Variables 에 아래 3개가 있어야 합니다:

```
DATA_GO_KR_API_KEY=...   # 공공데이터포털 통합키 (API 7개 공용)
BIZINFO_API_KEY=...      # 기업마당
YOUTH_CENTER_API_KEY=... # 온통청년
```

## 문제가 생기면

- **`DATA_GO_KR_API_KEY` 를 넣었는데 "서비스 키 오류" 나옴**
  → Decoding 버전 말고 **Encoding 버전** 을 복사했는지 확인. 복사 시 앞뒤 공백도 제거.

- **활용신청이 거절됨**
  → 활용목적을 더 구체적으로 (예: "정부 지원 정책 정보를 수집하여 사용자에게 맞춤 알림을 제공하는 웹 서비스 운영") 작성.

- **키는 있는데 403 / 401 응답**
  → 신청 상태가 "승인대기" 인지, "운영계정" 이 아닌 "개발계정" 인지 확인. 개발계정이면 하루 10,000건 요청 제한이 있어요.
